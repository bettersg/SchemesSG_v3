# SchemesSG v3 Frontend

Next.js + TypeScript app.

## Commands

- `npm run dev` — local dev (APP_ENV=development)
- `npm run build:staging` — staging build
- `npm run build:prod` — production build

## Environment

Download `.env.*` files from Google Drive and place in `frontend/`. Never commit them.

## Deployment

- `stg` branch → auto-deploys to `schemessg-v3-dev.web.app`
- `main` branch → auto-deploys to `schemes.sg`

## URLs

- Staging: https://schemessg-v3-dev.web.app
- Production: https://schemes.sg
